# Code of Conduct

SAGE follows the Contributor Covenant Code of Conduct.

We expect participants to be respectful, constructive, and considerate in issues, pull requests, discussions, reviews, and project communication.

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to project maintainers or organization owners. Reports will be reviewed with appropriate confidentiality and care.

Project maintainers may remove, edit, or reject contributions and interactions that do not align with this Code of Conduct.
